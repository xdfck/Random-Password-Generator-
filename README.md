# Random-Password-Generator-
GUI-приложение «Random Password Generator» с возможностью настройки параметров пароля, сохранением истории и Git
Автор: [Ермаков Дмитрий Евгеньевич]

Графическое приложение для генерации случайных паролей с возможностью настройки параметров (длина, типы символов), сохранения истории и загрузки ранее созданных паролей. Данные хранятся в формате JSON. Проект управляется через Git.

 Возможности

- Настройка длины пароля с помощью ползунка.
- Выбор типов символов: цифры, буквы (верхний/нижний регистр), спецсимволы.
- Генерация пароля по клику на кнопку.
- Отображение истории в таблице.
- Сохранение и загрузка истории из файла `history.json`.
- Проверка корректности ввода (минимальная/максимальная длина).

 Примеры использования

1. Запустите `main.py`.
2. Установите нужную длину пароля с помощью ползунка.
3. Выберите необходимые типы символов чекбоксами.
4. Нажмите «Сгенерировать» — пароль появится в поле и добавится в историю.
5. История сохраняется автоматически и загружается при следующем запуске.

 Технологии

- Python 3.x.
- Tkinter — для интерфейса.
- random — для генерации.
- json — для хранения данных.
